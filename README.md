# Tally Automation

## Overview

Tally Automation is a Python-based automation project designed to simplify accounting workflows in Tally. It helps reduce manual data entry by automating ledger creation and voucher processing through structured data inputs.

## Features

* Automated ledger creation
* Payment voucher processing
* Receipt voucher processing
* Single voucher entry automation
* Structured workflow for efficient accounting operations

## Technologies Used

* Python
* Pandas
* Requests
* BeautifulSoup4
* Tally XML Integration

## Project Structure

```text
ledger_creation.py
payment_vouchers.py
receipt_vouchers.py
single_voucher.py
```

## Getting Started

1. Install Python 3.x
2. Install the required dependencies.
3. Ensure Tally is running with XML integration enabled.
4. Execute the required Python script based on the desired operation.
